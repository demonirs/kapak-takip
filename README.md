# Kapak Takip Temiz Kurulum

1. Supabase SQL Editor içinde `supabase/migrations/001_schema.sql` dosyasını çalıştır.
2. Vercel Environment Variables içine şunları ekle:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
3. GitHub'a bu klasörün içeriğini yükle.
4. Vercel'de Root Directory boş veya proje kökteyse `./` kalsın.
5. Deploy et.
